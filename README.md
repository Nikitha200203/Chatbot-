# Mental Health Support Chatbot
- This is a simple Python-based chatbot designed to provide comfort and guidance to individuals experiencing depression, anxiety, or panic attacks. The chatbot uses predefined responses to engage in conversations that aim to support users during difficult times.

# Features
- Supportive Conversations: The chatbot offers comforting messages based on user input related to depression, anxiety, sadness, loneliness, and panic attacks.
- Simple Interface: A command-line interface where users can interact with the chatbot.
- Empathetic Responses: The chatbot uses a collection of reflective responses to help users feel heard and supported.
